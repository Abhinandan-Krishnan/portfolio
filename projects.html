My Projects

Autonomous multi-robot system for scalability in warehouse applications

It is a proof of concept for using collaboration between mobile robots to increase scalability in warehouses. Currently, an omnidirectional robot is developed with self localization and navigation capabilities to autonomously navigate to any specified location.

Trajectory following differential drive robot

Unicycle based autonomous differential drive robot with self localization using optical encoders and navigation capabilities. The robot can self locate and traverse to any specified location autonomously.

Switch Easy

Switch Easy is an affordable and modular plug and play device that attaches to any fan regulator and instantaneously turns it into a “smart” fan which automatically changes the speed to the user’s comfort based on the temperature and provides an app to remotely control the fan with and/or without internet.

Sound localization for navigation applications

Implementation of sound localization algorithm based on time difference of arrival (TDOA) approach, currently working on implementing it on the hardware. The algorithm can detect the origin of a sound with a ±5cm error and with a precision of over 95% (simulation results).

Smart intermediate e-mail translation device for rural applications

An intermediate translation software that receives the bank account mails, government scheme automated mails etc, processes and redirects it to the recipient’s 2G device as a text message(s) in regional languages.
